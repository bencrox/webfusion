WHO WE ARE?

Web Fusion Technology is spin off from Transcom Electronics Limited in 2008,  to focus up on realtime system and middleware platform API design. 

Managing Director Mr Ben Lee has in depth knowledge from embedded system, firmware programming to mobile and mesh network implementation. He is also a pioneer in using server side javascript for large scale control system, making non-blocking i/o stack and realtime web interface.

We serve large scale service companies such as ISS Facility Service Limited, Adams Security Limited, Pan Asia Security Limited, Compass Group and Government sectors such as HK Post Office, Environment and Hygiene Department.

Since 2012 Web Fusion has outreached to ShangHai and started providing consultancy to the emerging human resource intensive service companies.

We help home/office automation product designer to refine their web and mobile interface, and still integrated with traditional phone system so as to make easy adaption among existing workforce. We are proud of making systems that even elderly can remain well connected.
